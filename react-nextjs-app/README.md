# React Next.js App

This project is developed using React and Next.js.

## Installation

```shell
npm install
```

## Running the app

```shell
npm run build
npm run start

or

npm run dev
```

### Directory Structure

```text
.
├── app
│   ├── page.tsx
│   └── stopwatch.tsx
|   └── time.tsx
|   └── metrics.tsx
|-- .env
├── package-lock.json
├── package.json
├── README.md
```

```text
/app - Contains the React components
/.env - Environment variables
/package-lock.json - NPM lock file
/package.json - NPM package file
```
